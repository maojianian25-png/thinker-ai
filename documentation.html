Thinker AI - User Documentation

AI-Powered SEO Content Generator - PHP Web Application

Version: 1.0 | Platform: CodeCanyon

Overview

Thinker AI is a powerful PHP web application that generates high-quality, SEO-optimized content using OpenAI's GPT models. It features a clean admin dashboard, multi-language support, and flat-file storage (no database required).

Key Features

  • AI-powered content generation (articles, blog posts, product descriptions, etc.)
  • Multi-language support (English, Chinese, Japanese, Korean, Spanish, French, German, more)
  • Keyword-optimized SEO content
  • Multiple tone options (Professional, Conversational, Persuasive, Informative, Creative)
  • Clean, modern admin dashboard
  • Secure login system
  • Content history management
  • Export and copy content
  • No database required (flat-file JSON storage)
  • Responsive design (works on desktop & mobile)

System Requirements

  • PHP 7.4 or higher
  • cURL extension enabled
  • JSON extension enabled
  • OpenAI API key (get one at platform.openai.com)

Installation

1. Upload Files

Upload all files to your web server via FTP or cPanel File Manager. The files can be placed in any directory (e.g., public_html/thinker-ai/ or a subdomain).

2. Set Permissions

Make sure the storage/ directory is writable by the web server:

chmod 755 storage/
# or
chmod 777 storage/

3. Run Installation Wizard

Navigate to your installation URL in a web browser:

https://yourdomain.com/thinker-ai/install.php

Follow the on-screen wizard to set up your admin account and OpenAI API key.

4. Start Generating Content

After installation, you'll be redirected to the admin dashboard. Navigate to "Generate Content" to create your first AI-powered article!

Usage Guide

Generating Content

  1. Go to Generate Content in the sidebar
  2. Enter a topic (e.g., "Benefits of Remote Work")
  3. Optionally add keywords separated by commas (e.g., "remote work, productivity, work-life balance")
  4. Select content type, language, and tone
  5. Click "Generate with AI"
  6. Wait 15-30 seconds for the AI to create your content
  7. Copy or export the result

Managing Content

All generated content is automatically saved. Use Content History to view, search, and manage your generated pieces. Each piece can be viewed, printed, or deleted.

Troubleshooting

OpenAI API Error: Ensure your API key is correct and has sufficient credits. Go to Settings to update your key.
Blank Page / 500 Error: Make sure your PHP version is 7.4+ and the required extensions (cURL, JSON) are enabled. Check file permissions on the storage/ directory.
